# 제주교육마루 — 한국교육학회 제주지회 뉴스레터

> 한국교육학회 제주지회 뉴스레터 웹사이트
> 🔗 https://kera-jeju.github.io/newsletter-jeju/

## 개요

한국교육학회 제주지회에서 매년 발간하는 뉴스레터를 웹사이트로 제작·배포하는 프로젝트입니다.
`build.py` 스크립트가 원고 파일을 읽어 정적 HTML 웹사이트를 자동 생성합니다.

- **첫 발간**: 2025년 10월 15일 (재창간호)
- **배포**: GitHub Pages (push하면 자동 배포)
- **기술**: Python + 정적 HTML/CSS/JS

## 매년 새 호 발간 방법

### 1단계: 원고 수집

1. 필진에게 **구글 문서 템플릿**을 메일로 전달
2. 필진이 구글 문서로 작성 후 **메일로 회신** (이미지는 별도 파일 첨부)
3. 편집위원이 수합하여 구글 드라이브 폴더에 정리

구글 드라이브 원고 폴더 구조 (예: `G:\내 드라이브\2026 제주지회 뉴스레터`):

```
2026 제주지회 뉴스레터/
├── 00_발간사/          ← 회장 발간사
├── 01_주론/            ← 주론 원고
├── 02_시론/            ← 시론 원고
├── 03_제주교육소식/    ← 제주교육소식 원고 (여러 건)
├── 04_캠퍼스 네트워크/ ← 캠퍼스별 원고
├── 05_회원 동정/       ← 회원 소식
├── 06_제주지회 활동 소개/ ← 활동 관련
└── 배너_이미지/        ← 배너, 로고 등
```

### 2단계: 원고 서식

필자에게 아래 서식으로 원고를 요청합니다:

```
제목: (글 제목)
필자: (이름)
소속: (소속 기관)
직함: (교수/교사/박사과정 등)

---

(본문 내용)

---

참고문헌
홍길동(2024). 논문제목. 학술지명, 1(2), 1-10.
```

- 프로필 사진: `필자이름_사진.jpg`로 별도 첨부
- 본문 이미지: `[이미지: 파일명.jpg]`로 위치 표시, 이미지 파일은 별도 첨부

### 3단계: 빌드

```bash
# Claude Code에서 실행
python build.py --src "G:\내 드라이브\2026 제주지회 뉴스레터" --out ./2026
```

> **원고 형식별 빌드 방법**
> - **구글 문서(.gdoc)**: build.py가 Google Drive API를 통해 문서 내용을 읽어 변환
> - **노션 내보내기**: `python build.py --src /path/to/notion-export --out ./2026` (2025년 방식)
>
> 빌드에 문제가 있으면 Claude Code에 "뉴스레터 빌드해줘"라고 요청하면 처리해줍니다.

### 4단계: 메인 페이지 업데이트

`index.html`에 새 호 카드를 추가합니다:

```html
<!-- 기존 카드 위에 추가 -->
<a class="issue-card" href="2026/">
  <div class="issue-info">
    <h3>제2호</h3>
    <p>2026년 10월 발행</p>
  </div>
  <span class="issue-arrow">&rarr;</span>
</a>
```

### 5단계: 배포

```bash
git add -A
git commit -m "2026년 뉴스레터 추가"
git push
```

push 후 2~3분이면 GitHub Pages에 자동 반영됩니다.

## build.py 구조

| 영역 | 설명 |
|------|------|
| `SECTIONS` | 뉴스레터 섹션 정의 (제목, 파일 매핑, 하위 구조) |
| `CSS` | 전체 스타일시트 |
| `INLINE_JS` | 3단계 네비게이션 (탭 → 카드 목록 → 개별 글) |
| `build_html()` | HTML 템플릿 조립 |
| `build_section_html()` | 섹션별 HTML 생성 |
| `md_to_html()` | 마크다운 → HTML 변환 |
| `build()` | 메인 빌드 함수 + 후처리 |

### 주요 후처리 (build 함수 내)

- 각주 상호 링크 (섹션별 고유 ID)
- 참고문헌 내어쓰기 (hanging indent)
- 표/그림 캡션 스타일링
- 프로필 사진 → author-card 변환
- 소속 표기 통일 (제주대→제주대학교)
- 노션 마크다운 잔여물 제거
